Quick Lua Syntax Guide

A short reference for the sandbox above.

Basics

  • local declares a scoped variable — always prefer it over globals
  • .. concatenates strings, not +
  • ~= means not-equal, not !=
  • Blocks end with end, except repeat, which ends with until

Tables

  • Tables are Lua's only data structure — used as arrays, maps, and objects
  • #t gives the length only for sequential arrays
  • pairs(t) iterates all keys, ipairs(t) iterates the numeric sequence

Lua Sandbox – FAQ

What is a Lua sandbox online?+
A Lua sandbox online is a website where you can write and run Lua code without installing Lua on your machine. This sandbox runs a real Lua 5.4 virtual machine compiled to JavaScript, so your script executes entirely inside the browser tab — nothing is sent to a server.
Is my Lua code executed on a server?+
No. Most online Lua compilers send your code to a backend server, run it there, and stream the result back. This tool instead runs a Lua interpreter directly on your device, so your code never leaves your browser and there is no queue or server timeout.
What is the difference between a Lua visualizer and a plain Lua compiler?+
A plain compiler only shows what your script prints to the console. A Lua visualizer additionally inspects the program's final state — every variable, table, and function your script created — and displays it as a structured list, so you can see what your code actually built without adding print statements for every value.
Can I use this as a Lua visual editor for larger scripts?+
Yes. The editor includes synced line numbers, monospace formatting, and instant error feedback with the offending line reported in the output panel, which covers the core needs of a Lua visual editor for scripts up to a few hundred lines.
Does this Lua sandbox work for Roblox Luau scripts?+
This sandbox runs standard Lua 5.4, not Luau. Roblox-specific globals like game, workspace, or Instance.new are not available here since they only exist inside the Roblox engine. Use it to test plain Lua logic, algorithms, and data structures before porting them into Roblox Studio.
